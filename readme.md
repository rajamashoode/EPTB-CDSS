**Supervisor:** Dr. Yasser MSA Alkharaman  
**Institution:** COMSATS University, Abbottabad Campus

### Project Overview
This tool is designed to assist clinicians in evaluating Extra-Pulmonary Tuberculosis (EPTB) cases based on the latest WHO Guidelines. It checks for:
- Correct Regimen Duration
- Dosage Weight-Banding
- Drug-Drug Interactions (HIV/Diabetes)
- Renal Adjustments

### How to Run Locally
1. Install requirements: `pip install -r requirements.txt`
2. Run the app: `streamlit run eptb_tool.py`
